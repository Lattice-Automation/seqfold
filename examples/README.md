# examples

- **`known_structures.fasta`** — real nucleic-acid sequences with known
  secondary structure (yeast tRNA-Phe, E. coli 5S rRNA, tRNAs, and structured
  RNA/DNA oligos from seqfold's regression set). These have genuine hairpins,
  stems, and multi-branch junctions, so they fold into real structures — unlike
  random sequences. Each header lists the length, a reference free energy
  (UNAFold / literature), and the energy seqfold computes. Used for benchmarking
  and as realistic folding examples.

- **`bench.rs`** — wall-clock benchmark over a subset of the above.
  Run `cargo run --release --example bench` (add `RAYON_NUM_THREADS=1` to
  measure single-threaded).

- **`dna.csv` / `rna.csv`** — regenerated by `tests/fold_examples.py`; compare
  seqfold's free energy against UNAFold across the regression sequences.
